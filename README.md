# Linux & Shell Scripting Assignment

## Overview
This assignment covers fundamental Linux concepts, shell commands, and scripting techniques. It demonstrates how to interact with the Linux terminal, execute commands, and automate tasks using shell scripts.

## Topics Covered
- Linux Basics: Navigating the filesystem, managing files and directories.  
- Shell Commands: Networking commands like `ping`, `ip a`, file operations, and process management.  
- Shell Scripting: Writing scripts to automate tasks, including creating executable `.sh` files and using variables, loops, and conditionals.

## Files in this Repository
1. `backup`
- Automates backing up specified directories to a target location.
- Logs the date and time of each backup.

2. `download`
- Automates downloading files from the internet using URLs.
- Can handle multiple files and logs the download status.

3. `monitor`
- Monitors system or network status (e.g., CPU usage, memory usage, connectivity).
- Outputs information to the terminal or logs for analysis.
